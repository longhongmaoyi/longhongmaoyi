📖 How To Use This Admin Panel

📦 To Add a Product

  1. Open your Google Sheet
  2. Add a new row at the bottom
  3. Fill in: name, price, category, emoji, image
  4. Click Refresh in this panel
  5. Product appears on website ✅

📩 To Reply to an Inquiry

  1. Go to the Inquiries tab
  2. Click on the inquiry
  3. Click "Reply on WhatsApp" or "Send Email"
  4. Mark as Replied when done ✅

✏️ To Update Website Text

  1. Go to Page Content tab
  2. Edit the text you want to change
  3. Click Save Changes
  4. View Website to confirm ✅

🔗 To Connect Google Sheet

  1. Publish your sheet as CSV
  2. Copy the CSV URL
  3. Go to Settings tab
  4. Paste URL → Save & Connect ✅
